#Ride The Bus - State Machine & Probability Project

Ride The Bus is fun project that I decided to create after noticing that you can play with mathematically optimal choices. I had fun making this and learned a lot.

## Features

- **Deck Management** Simulates a 52-card deck using a Python dictionary structure that dynamically tracks card count and removes card in real time.
- **State-Driven Game Loop** Utilizes state machine architecture that tracks player progression through four rounds without deep code nesting.
- **Tie Handling** Detects card ties and resets the round while dynamically handling the deck.
- **End Conditions** Monitors the deck and win conditions to detect game end state.
- **Win-Loss-Card-Analytics** Tracks current card, win, loss, number of cards it took to win, cna current card count.
- **COMING SOON** Ride The Bus will soon be able to calculate probabilties and mathematiclly optimal card choices, providing users with the best path to winning.
